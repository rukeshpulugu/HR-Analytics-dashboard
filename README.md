# HR Analytics Dashboard

## Project Overview

This project analyzes employee attrition and workforce trends using Power BI. The dashboard helps identify factors influencing employee retention, job satisfaction, and workforce performance.

## Tools Used

* Power BI
* DAX
* Data Visualization
* HR Analytics

## Key Metrics

* Total Employees
* Attrition Count
* Average Age
* Average Monthly Income

## Dashboard Features

* Employee Attrition Analysis
* Department-wise Employee Distribution
* Job Role Analysis
* Gender Distribution
* Salary Analysis
* Age Distribution
* Overtime vs Attrition Analysis
* Interactive Slicers and Filters

## Business Insights

* Identified departments with higher employee attrition.
* Analyzed the impact of overtime on employee turnover.
* Compared employee distribution across job roles.
* Evaluated salary and demographic trends.
* Provided workforce insights to support HR decision-making.
